## рџ“Њ Overview

**Serve Smart** is a computer vision project built using [Ultralytics YOLOv8](https://github.com/ultralytics/ultralytics). It trains a high-accuracy object detector on a military dataset and produces YOLO-format prediction files ready for submission.

|---|---|
| Model | YOLOv8m (medium) |
| Classes | 12 military object categories |
| Input Size | 640Г—640 |


### 2. Install Dependencies

```bash

```

Or manually:

```bash

```

> **GPU Recommended:** For best results, use a CUDA-enabled GPU.  
> On Google Colab: `Runtime в†’ Change runtime type в†’ T4 GPU`

Place your dataset inside `data/military_object_dataset/` following the structure above, or provide a Google Drive URL directly in the notebook (Cell 7).

Open `Serve_Smart_YOLO_Pipeline.ipynb` and run cells top to bottom.
